I Learned Css

CSS is the creative way to make your page with designes and colors. Also you can make all your margins spacing and all from the css page. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.
